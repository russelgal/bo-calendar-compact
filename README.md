# БО Компактная шахматка — расширение для Chrome/Brave

Компактный вид шахматки «Бронируй Онлайн» — скрывает заголовки категорий, строки «Без номера», итоги. Оставляет только данные номеров.

Работает только на `lk.bronirui-online.ru`.

## Установка

1. Скачать или клонировать этот репозиторий
2. Открыть в браузере: `chrome://extensions`
3. Включить **Режим разработчика** (переключатель в правом верхнем углу)
4. Нажать **Загрузить распакованное расширение**
5. Выбрать папку с расширением
6. Закрепить иконку в панели браузера (пазл → булавка)

## Использование

- **Левый клик** по иконке — вкл/выкл компактный вид (badge «ON» когда активно)
- **Правый клик** по иконке → **Настройки** — попап с обновлением правил

## Обновление правил

Правила скрытия хранятся в `presets.json` в этом репозитории. Расширение проверяет актуальность через GitHub API:

- Если правила актуальны — кнопка «Актуально» (неактивна)
- Если есть обновление — кнопка «Обновить правила» (активна)

Если сайт «Бронируй Онлайн» изменит вёрстку — достаточно обновить `presets.json` в репозитории, и все пользователи получат новые правила через настройки.

## Обновление расширения

После изменения кода расширения (не правил):

1. `git pull` или скачать заново
2. Открыть `chrome://extensions`
3. Нажать кнопку обновления (⟳) на карточке расширения
4. Перезагрузить страницу календаря
